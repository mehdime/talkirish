'Cad é an' and 'Cén' mean exactly the same thing. If you lokk closely you will see that 'cén' is 'cad é an' made into one word. If you see below, the capital letters show the letters CÉN ('Cad' has been abbreviated if you like to 'C', é stays the same and 'an' has been abbreviated to 'N'')

C - ad

É

a-N
