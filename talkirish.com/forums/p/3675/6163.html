    Seán,

    Sorry to hear that it's not been straightforward for you. Let's try and troubleshoot the basics. First open the album in iTunes. Then select any file and right-click it. You should get an option to 'Get Info'. Click that.

    In the dialog box that's displayed, you should be able to select a tab called 'lyrics'. Click this. It will display the course lyrics - so this first step will confirm that the course has imported correctly to iTunes.

    Let me know how that works out. I'll double check how Buntús Cainte performs on my iPod (at home!) this evening and reply again tomorrow.
